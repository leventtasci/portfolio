# Connect Serisi Konfigürasyon Sistemleri

Bentley Systems'in Türkiye distribütöründe geliştirdiğim konfigürasyon sistemleri hakkında buradan bilgi alabilirsiniz.

## Neden bir Konfigürasyon Kistemine İhtiyaç Var?

Çünkü
  - Kütüphaneler, civil cell'ler, yazılımlar gibi tüm kaynakları,
  - Stabil bir çalışma ortamı için gerekli ayarları,
  - Tüzel kişiliklerin yapısına ve kullanıcının rolüne göre
eşgüdümlü olarak dağıtılmasına ihtiyacı var.
Bu sayede herkes aynı kaynakları kullanarak projelerine odaklanır

```{toctree}
:hidden:
index
stucture
authorizers
contractors
```
